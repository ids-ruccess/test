## 프로젝트 시작할때는 그 당시 nest 최신버전을 지향합니다.

```
//step1
npm install -g @nestjs/cli

//step2: nest 최신버전 확인
nest --version

// update deependencies
npm i -g npm-check-updates

//step: 3 update package
ncu -u -f /^@nestjs/
```
